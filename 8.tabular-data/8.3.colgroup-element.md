# 8.3.colgroup element

HTML Table Column Group 요소 \(&lt;colgroup&gt;\) 는 표 내부의 칸의 그룹을 정의합니다.  
table의 자식요소로 caption요소보다 먼저 선언되야 하지만 생략가능.

```text
<table>
	<colgroup>
		<col>
		<col>
		<col>
	</colgroup>
	.
	.
	...
</table>
```
