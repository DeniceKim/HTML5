# 8.4.col element

HTML Table Column 요소 \(&lt;col&gt;\) 는 표 내부의 열을 정의하며 모든 공통 셀들의 공통 시멘틱을 정의하기 위해 사용됩니다.  
일반적으로 &lt;colgroup&gt; 요소 내부에 있습니다.  
단일요소로 종료 태그 없이 사용됩니다.

```text

<table>
	<colgroup>
		<col>
		<col>
		<col>
	</colgroup>
	.
	.
	...
</table>
```
